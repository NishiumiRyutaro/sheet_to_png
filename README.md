# Spreadsheet to png converter

- Python 3.7.5
- ImageMagick 7.0.9

## Deploy Heroku
1. Add Buildpack https://github.com/epicatization/heroku-buildpack-imagemagick-heif
2. Add Config Vars. DISCORD_TOKEN, SHEET_URL

## Commands
## /hello
Hello!

## /sheet (start) (end)
return the range from start to end as an image.
